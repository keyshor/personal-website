Other Projects

Internships

  • Research Internship at Nokia Bell Labs (Summer 2020). I worked on applications of deep reinforcement learning and imitation learning to classical regenerative stopping problems and studied the effectiveness of learning based solutions in logistics optimization.

  • SDE Internship at Amazon Web Services (Summer 2019). I interned with the Automated Reasoning Group at AWS. I worked on automatically discovering sinks of sensitive data in Java code. Functions known as sinks are usually given as inputs to a taint analysis tool to check for security vulnerabilities in code. New techniques are needed to find such sinks since manual examination of code is close to impossible in large codebases. My project was on applying machine learning to identify sinks in the codebase of AWS storage services.
